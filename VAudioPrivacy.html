Privacy Policy

VAudio Elite App - How your data is handled and protected.

Effective date: 1 Aug 2026  ·  Version: 1.0.0.0  ·  Publisher: Hasnain Butt Akhtar

1. Overview

VAudio Elite is a Windows desktop media player with integrated spatial audio enhancement, equalizer, cinematic video filters and reactive visualizers.

VAudio Elite performs all audio and video processing locally on your device and does not transmit personal data, media files or playback activity to Hasnain Studio X.

We do not sell, share, or monetize user data.

2. Data We Do NOT Collect

VAudio Elite does not collect or send the following to the developer:

Identity Data

Name, email, phone number, or personal identity data.

Media Content

Your music and video files, library metadata, and playback history are not uploaded to our servers.

Telemetry

No analytics, ad tracking, or third-party telemetry SDKs are used.

Sensitive System Data

No password, credential, keychain, or hidden account data is collected.

3. Data Stored Locally

Local Library Index

The folders you choose to scan and the metadata read from your media files are cached locally so the Library opens quickly.

App Settings

Volume, repeat / shuffle, equalizer state, enhancement presets, last-played item and other preferences are saved to a settings file under your Windows user profile.

Trial / License State

A local file stores the trial start date and Microsoft Store entitlement state so the app can honour your purchase across launches.

Browser Session Data

If you sign in to YouTube or any other site inside the Browse page, the resulting cookies and login state are stored in WebView2's sandbox under your Windows user profile, exactly as a browser would.

4. Network Activity

Browse Page

The Browse page contacts youtube.com (and any other site you visit) the same way Microsoft Edge would. Data sent to those sites is governed by their own privacy policies, not by VAudio Elite.

Direct Stream URLs

When you paste a direct media URL or use Enhanced YouTube playback, VAudio Elite fetches the media stream over HTTPS for playback. The stream is not retained, recorded or shared.

No Background Calls

VAudio Elite does not phone home, ping update servers, or send usage pings of any kind.

5. Microsoft Store Purchases

VAudio Elite is sold via the Microsoft Store as a one-time durable purchase. Payment, billing and transaction processing are handled entirely by Microsoft under their own policies. VAudio Elite does not process or store payment card details.

After purchase, VAudio Elite checks the Microsoft Store licence locally via the public Windows.Services.Store API and unlocks features. No additional identifying information is sent to Hasnain Studio X.

6. App Permissions

VAudio Elite requests only the minimum permissions required to run core media features: read access to media folders you choose, the internet client capability used by the embedded browser, and access to the system audio output device for playback and enhancement.

7. Children's Privacy

VAudio Elite is a general-audience media player intended for personal use and is not directed to children under 13.

8. Policy Changes

If this policy changes, we will update the effective date and version on this page.

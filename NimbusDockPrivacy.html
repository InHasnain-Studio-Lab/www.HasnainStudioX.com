Privacy Policy

NimbusDock - How your data is handled and protected.

Effective date: 18 May 2026  ·  Version: 1.0.1.0  ·  Publisher: Hasnain Butt Akhtar

1. Overview

NimbusDock is a Windows desktop app that displays a floating glassmorphic dock for launching pinned applications, with live system metrics, weather, and tile status for selected installed apps.

NimbusDock performs all processing locally on your device and does not transmit personal data to Hasnain Studio X.

We do not sell, share, or monetize user data.

2. Data We Do NOT Collect

NimbusDock does not collect or send the following to the developer:

Identity Data

Name, email, phone number, or personal identity data.

App Usage Telemetry

No analytics, ad tracking, click-tracking, or third-party telemetry SDKs are used.

App Inventory Uploads

Your list of pinned apps, launch history, or shortcut targets is never uploaded.

Sensitive System Data

No passwords, credentials, keychain entries, or hidden account data is collected.

3. Data Stored Locally

Dock Settings

Pinned app paths, shape, accent, icon size, position, and workspace presets are stored as a single JSON file in %LOCALAPPDATA%\NimbusDock\settings.json.

Launch History

Local 14-day rolling launch counts power the "recents auto-fill" feature. The history never leaves your device and is auto-pruned when files no longer exist.

License State

Microsoft Store trial / Full license state is read from the Store on each launch and cached in memory only.

4. Network Use

Weather

If the weather widget is enabled, the app queries the public Open-Meteo API (api.open-meteo.com) using approximate coordinates resolved via ipapi.co. No personal data is sent; no API key is required. Both can be disabled in Settings.

Microsoft Store

Trial / purchase state is read via the Windows Store APIs from your locally signed-in Microsoft account. NimbusDock does not see your account details.

5. Microsoft Store Purchase

NimbusDock is a paid Microsoft Store app with a Store-managed free trial. Payment and transaction processing are handled by Microsoft Store under Microsoft policies.

NimbusDock does not process, store, or transmit payment card details.

6. App Permissions

NimbusDock declares only two MSIX capabilities:

internetClient — required for the optional weather feature.
runFullTrust — required to launch the Win32 desktop applications you pin to the dock.

No location, contact, camera, microphone, photos, or background activity permissions are requested.

7. Auto-Start & Tray

NimbusDock writes a single entry to HKCU\Software\Microsoft\Windows\CurrentVersion\Run so it launches with Windows. The entry can be removed at any time from Settings → Start with Windows or the tray menu.

8. Children's Privacy

NimbusDock is a general-productivity desktop utility and is not directed to children under 13.

9. Policy Changes

If this policy changes, we will update the effective date and version on this page.
